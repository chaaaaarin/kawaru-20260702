# 🎁 AGENTS.md（SEOルール）

SEOループを回すエージェントに読ませる AGENTS.md のSEO運用ルール雛形。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: リポジトリ直下の AGENTS.md にコピーして、自分のサイト事情に合わせて調整する。

```text
# AGENTS.md — SEO運用ルール

## 役割
このリポジトリで動くエージェントは、SEOループ
（収集 -> 分析 -> 執筆 -> 評価 -> 改稿 -> 監視）の一部を担う。

## 執筆ルール
- 検索意図を最初に定義してから書く
- 誇張・断定保証をしない（「必ず上位表示」等はNG）
- 事実には裏付けを付ける。裏が取れない主張は書かない or 注記する
- E-E-A-T（経験・専門性・権威性・信頼性）を満たす

## 品質ゲート（停止条件）
- Reviewer の必須指摘がすべて解消するまで公開しない
- 公開は人間の最終確認後

## メモリ / ログ
- 作業のたびに LOG に「対象・変更・結果指標」を追記する
- セッション開始時は直近の LOG を読んでから着手する

## やってはいけないこと
- キーワードの不自然な詰め込み
- 他サイトの文章の複製
- 出典のない数値・価格・日付の断定
```

**ポイント**: 「品質ゲートを通るまで公開しない」を明記すると、自動化しても質が崩れない。
