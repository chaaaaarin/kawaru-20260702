# 🎁 SEO Loop 設計テンプレート

SEOを「1回書いて終わり」ではなく、自律的に回り続けるループとして設計するための穴埋めテンプレート。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: そのままコピーして、対象サイト・指標・停止条件を埋める。エージェントの初期設定やAGENTS.mdの隣に置く。

```text
# SEO Loop 設計テンプレート

## 1. ループの目的（Goal）
- 対象サイト / セクション:
- 改善したい指標:（例: 対象KW群の平均掲載順位 / 対象ページのCTR）
- 目標値と期限:

## 2. トリガー（いつ回すか）
- [ ] cron（例: 毎日 06:00）
- [ ] Search Console のアラート（順位 / CTR 下落）
- [ ] 新規KW発見（他ループからの通知）
- [ ] 手動

## 3. ループ本体（Act -> Observe）
1. Research: 対象KWのSERP・競合上位を分析し、勝ち要件を出す
2. Writer: 要件に沿って新規作成 or 該当箇所を改稿
3. SEO Checker: SEO/GEOスコア・内部リンク・構造化データを検査
4. Reviewer: 事実・E-E-A-T・読みやすさを点検
5. Publisher: CMSへ反映（下書き / 公開）

## 4. 停止条件（Stop condition）
- [ ] 品質スコア >= (閾値)
- [ ] Reviewer の必須指摘がすべて解消
- [ ] 目標順位に到達 or 上限イテレーション（例: 5回）に到達

## 5. 記録（Memory / Log）
- 実行日時・対象・変更点・結果指標を LOG に追記
- 次回はこの LOG を読んでから開始する

## 6. ガードレール
- 公開はレビュー後（人間の最終確認を挟む工程を明記）
- 1回あたりのコスト上限:
```

**ポイント**: 停止条件を必ず数値で決めておくと、無限ループ・コスト暴走を防げる。
