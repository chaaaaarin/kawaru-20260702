# 🎁 キーワードリサーチ プロンプト集

検索意図の分類から記事ブリーフ生成まで、Research Agent にそのまま渡せる4本のプロンプト。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: {{ }} の箇所を自分のキーワードに置き換えて、AIエージェントに投げる。

```text
# キーワードリサーチ プロンプト集

## 1. 検索意図の分類
以下のキーワード群を、検索意図（Know / Do / Go / Buy）で分類し、
各意図で狙うべきコンテンツ形式を表にして。
キーワード: {{keywords}}

## 2. 競合ギャップ抽出
対象KW「{{keyword}}」で上位10件の見出し構成を要約し、
多くのページが共通でカバーしているサブトピックと、
逆に手薄なサブトピック（ギャップ）を分けて列挙して。

## 3. ロングテール展開
シードKW「{{keyword}}」から、検索意図が明確でコンバージョンに近い
ロングテールKWを20個、意図の注釈つきで出して。

## 4. コンテンツブリーフ生成
KW「{{keyword}}」の記事ブリーフを作成:
想定読者 / 検索意図 / 推奨文字数 / H2・H3構成案 /
含めるべきエンティティ / 内部リンク候補 / FAQ 5件。
```

**ポイント**: 出力はそのまま公開せず、必ず一次データ（実際のSERP）で裏取りしてから使う。
