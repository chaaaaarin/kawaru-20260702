# 🎁 記事レビュー チェックリスト

Reviewer / SEO Checker が公開前に毎回通す、E-E-A-T × SEO のチェックリスト。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: 記事ごとにコピーして、各項目をチェックしてから公開する。

```text
# 記事レビュー チェックリスト（SEO × E-E-A-T）

## E-E-A-T
- [ ] Experience: 一次体験・実データ・具体例が入っているか
- [ ] Expertise: 記述が正確で、専門用語の使い方が適切か
- [ ] Authoritativeness: 主張に信頼できる裏付け / 出典があるか
- [ ] Trust: 誇張・断定保証がなく、透明性があるか（最重要）

## 検索意図
- [ ] タイトル・導入が検索意図に一致しているか
- [ ] 読者の「次の疑問」に本文で答えているか

## SEO 技術
- [ ] タイトル / メタディスクリプションが適切（長さ・KW）
- [ ] 見出し階層（H1 -> H2 -> H3）が論理的
- [ ] 内部リンク（関連記事・回遊）が張られている
- [ ] 構造化データ（schema）が適切
- [ ] 画像 alt / 代替テキスト

## 可読性
- [ ] 一文が長すぎない / 箇条書きと散文のバランス
- [ ] 結論が先、根拠が後

## 公開前
- [ ] 事実確認済み（未確認は注記 or 削除）
- [ ] 人間の最終レビュー完了
```

**ポイント**: Trust（信頼性）が最重要。誇張表現を1つでも見つけたら中立表現に直す。
